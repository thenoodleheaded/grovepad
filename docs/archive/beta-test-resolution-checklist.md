## Final verification gate

- `npm run check:full` — passed: production TypeScript/Vite build, lint, 60 test files / 661 tests, documentation links, Supabase migration parse, and database-test syntax.
- `npm run test:focus` — passed: 2 files / 13 focused reorder and sizing tests.
- `git diff --check` — passed with no whitespace errors.
- Browser checks used a fresh isolated origin and public UI actions. Verified picker/menu semantics, rapid selection/board framing, zero native canvas scroll, keyboard Circuit wiring and pressed state, Focus isolation/exit, dependency ownership copy, blank Table copy, and populated nested-canvas deletion disclosure. Scratch widgets and the test connection were removed through Undo; no scratch board content remains.
- Not manually certified here: physical touch/trackpad behavior, VoiceOver/NVDA announcements, a live local-model generation, a real hanging remote HTTP endpoint, signed-in cloud synchronization, or destructive snapshot confirmation on a disposable user profile. Deterministic tests cover the changed logic for these paths; they remain device/service integration risks rather than known unresolved report defects.
- Production build retained the existing warning for deferred chunks over 500 kB; no new build failure or eager-load regression was reported by Vite.
